# Smart Parking Slot Detection

An AI-powered Smart Parking Slot Detection and Monitoring System that automatically detects parking slot occupancy using a custom-trained YOLOv8 model and visualizes parking information through an interactive React dashboard.

The system processes parking lot video footage, identifies the occupancy status of each parking slot, and provides real-time visualization using charts, heatmaps, slot maps, statistics, and interactive dashboards.

---

## Features

* Real-time Parking Slot Occupancy Detection
* Custom YOLOv8 Model (`parking82_best.pt`)
* Interactive React Dashboard
* 2D Parking Slot Visualization
* 3D Parking Slot Visualization
* Occupancy Heatmap
* Parking Statistics Dashboard
* Pie Charts and Summary Cards
* Interactive Parking Map using Leaflet
* Video Playback Integration
* JSON-based Parking Slot Mapping
* CSV and Excel Report Generation
